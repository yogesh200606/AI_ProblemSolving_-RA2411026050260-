# AI_ProblemSolving_-RA2411026050260-
🎮 Interactive Tic Tac Toe AI (Minimax Algorithm)
📌 Description

This project implements an interactive Tic Tac Toe game where a human player competes against an AI.
The AI uses the Minimax algorithm to make optimal decisions, ensuring it never loses.

🧠 Algorithms
Minimax Algorithm
Game Tree Search
⚙️ Features
Interactive gameplay (User vs AI)
AI makes optimal moves
Detects win, loss, and draw conditions
Simple and user-friendly interface
Restart option available
🚀 How to Run
👉 For Python Version:
Run:
python main.py
Enter positions (0–8) to play
👉 For Web Version:
Open:
index.html
Play by clicking on grid
